6.1 递归函数

1. 递归函数会一直调用自身的拷贝函数，同时返回值会返回给调用它的拷贝函数

2. 递归函数一定要有一个结束条件（让递归不再调用函数）

3. 递归函数如果使用不当，会让系统崩溃，很快的耗尽计算机资源

4. Python 允许的最大递归深度是 990 ~ 1000 之间的值

例子：

# 递归累加 1 + 2 + ... + n
def f(num):
    if num == 1:
        return num
    else:
        return num + f(num - 1)


# 调用递归函数
print(f(100)) # 5050
